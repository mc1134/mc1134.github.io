As a rule of thumb, always use lowercase attributes and always quote attributes, like <a href="this stuff">. Technically, <a href=this stuff> is acceptable, but in some cases the space will separate what you want to say.

If you want to use "double quotes" in an attribute, use single quotes for the attribute - <p title='He "typed" it'> - and vice versa - <p title="He 'typed' it">. Try not to blow your mind by trying to use both quotes and apostrophes in an attribute that requires quotes.

HTML STYLES

The default style for an HTML document is white background, black text with 12 pt. Times New Roman font (for paragraph displays)

Use the "style" attribute in the <body> tag: <body style="background-color:lightgrey">

The background color of this document is aquamarine. The syntax is: <body title="background-color:aquamarine">

In general, the syntax for using the "style" attribute is: style="property:value". Both of these are CSS values.

To change the color of the text, use the syntax: style="color:XXXXX" where "XXXXX" is a color.

This text is red.

To change the font of the text, use the syntax: style="font-family:XXXXX" where "XXXXX" is a font.

This font is Calibri.

To change the size of the text, use the syntax: style="font-size:XXX%" where "XXX" is a percentage (100% is normal size)

This font is twice as big as normal.

To change the alignment of the text, use the syntax: style="text-align:XXXXX" where "XXXXX" is an alignment (left, center, right).

This text is aligned to the right.

Headings, as well as paragraphs, can also have colored text, different fonts, different sized text, and different alignments.

Style Attributes:		Description:
background-color		Changes the background color of the text. Only usable in the <body> tag.
color				Changes the color of the text.
font-family			Changes the font of the text.
font-size 			Changes the size of the text.
text-align			Changes the alignment of the text.

HTML TEXT FORMATTING

Text can be made to be bold, important, italic, emphasized, marked, small, deleted, inserted, subscripts, superscripts, and more.

Text formatting elements are tags, such as <p><b>bolded text</b>not bolded text</p>

not bolded bolded

The <strong> tag is the same as the <b> tag except it adds "importance" to the text: Strong text, bold text

The <i> and <em> (italics and emphasized) tags are all italics, with <em> having more "importance" than italics: Italics, Emphasized

"Small formatting" defines "small" text: use the <small> tag: smaller, bigger

"Marked formatting" defines marked or highlighted text: use the <mark> tag: marked text (may not show up in some HTML versions)

"Deleted" text is text with a single strikethrough: use the <del> tag: I like blue red

"Inserted" text is underlined text: use the <ins> tag: I like red

Subscript formatting uses the <sub> tag: y0=2

Superscript formatting uses the <sup> tag: y'=x2y3-excos(y)

HTML QUOTATION AND CITATION ELEMENTS

The <q> tag produces a short quote, producing quotes around the <q> elements

The <blockquote> tag quotes a block of stuff and tabs it; kind of like how <p> and <pre> are related:

Jack and Jill Went up the hill To fetch a pail of water;

Use the <abbr> tag to abbreviate an acronym (usually with a title describing the abbreviation): <abbr title="World Health Organization">WHO</abbr>
Essentially the <abbr> tag takes a section of the paragraph text and applies the "title" attribute to only that section of text
The WHO is an organization of the United Nations.

Use the <address> tag to define contact information in a document, usually displaying the information in italics: <address>Written by Person. Phone number: 000-000-0000</address<>

Written by Person. Phone number: 000-000-0000

Use the <cite> tag to cite a work; also italicizes the cited segment (The Scream): <p><cite>The Scream</cite> by Edward Munch.</p>

The Scream by Edward Munch

Use the <dfn> tag to define a term. The <dfn> tag has many syntactical usages:

  • If the <dfn> tag is used on its own, the term is only italicized (it is still italicized in all cases): PI is 3.14159265358979323846264338327950288419716939937510582097494459 more or less
  • If the title attribute in <dfn> is present, then the item inside the tags is italicized and displays the title when the cursor hovers over it: <dfn title="3.141592654">PI</dfn>
  • If the title attribute in <abbr> is present and <abbr> is encapsulated in the <dfn> tags, then the item inside the tags does the exact same thing as the previous syntax <dfn title=""></dfn>, except in this case the writer of the HTML document would be specifying an abbreviation instead of just a definition

To add a comment in HTML, use the syntax <!-- ... -->, where the ellipses represent where your comment in the HTML code go

<!--[if IE 8]>
.... some HTML here ....
<![endif]-->
